# fdm-gallery

FDM 6 plugin for downloading image galleries using gallery-dl.

## Supported Sites

- Pixiv
- Twitter/X
- 微博
- Danbooru/Gelbooru
- Reddit
- Tumblr
- Instagram
- Unsplash
- And more...

## Building

Unix:
```bash
cd build && ./build-unix.sh
```

Windows:
```batch
build\build-windows.bat
```

## Installation

1. Build or download `fdm-gallery.fda`
2. Install in FDM via menu
